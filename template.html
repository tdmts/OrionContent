Terminal Window

Gebruik dit component voor command line instructies (Putty, Bash, CMD). De commando's zijn kopieerbaar via de knop die verschijnt bij hoveren, terwijl de prompt en output statisch blijven.

pi@raspi:~$ sudo apt-get update 
Hit:1 http://archive.raspberrypi.org/debian bullseye InRelease
Get:2 http://raspbian.raspberrypi.org/raspbian bullseye InRelease [15.0 kB]
Fetched 15.0 kB in 1s (15.9 kB/s)
Reading package lists... Done
pi@raspi:~$ whoami 
pi
pi@raspi:~$ clear

Config Editor

Gebruik dit component om wijzigingen in configuratiebestanden aan te geven. Groene regels zijn nieuw, oranje regels moeten aangepast worden. Er verschijnt een kopieerknop bij hoveren.

/etc/codesyscontrol/CODESYSControl_User.cfg
[SysCom]
Linux.Devicefile.1=/dev/ttyUSB0
Linux.Devicefile.2=/dev/ttyACM0
